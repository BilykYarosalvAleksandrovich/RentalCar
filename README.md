# RentalCar

Веб-додаток для оренди автомобілів, побудований на Next.js 14, TypeScript та Zustand.

## Основні функції

- Каталог автомобілів з пагінацією та фільтрацією на бекенді.
- Система "Обране" зі збереженням у LocalStorage.
- Детальна сторінка автомобіля з формою бронювання.
- Адаптивна верстка та інтерактивні сповіщення.

## Технологічний стек

- **Framework:** Next.js (App Router)
- **State Management:** Zustand
- **Styling:** Tailwind CSS
- **API Client:** Axios
- **Notifications:** React Hot Toast

## Встановлення та запуск

1. Клонуйте репозиторій.
2. Виконайте `npm install`.
3. Запустіть проєкт командою `npm run dev`.
4. Відкрийте [http://localhost:3000](http://localhost:3000).
